What is Scarlet Rat?

Scarlet Rat is a demonstration of Selenium Webdriver and static-site-generation using Python, HTML, CSS, and JS

It provides interesting results from websites throughout the internet and posts snippets them all in one place.

It is presumably like how a rat or a bird would make a nest.


How does Scarlet Rat work?

Scarlet Rat runs on Python and manipulates HTML CSS and JS as an output.

  1. Make a search term using a few words from a sample text.
  2. Open Chrome and search up the sample text.
  3. Select a random website and open it.
  4. Scan the HTML to make a list of lowest level Divs.
  5. Create a "block" using a Div with sufficient content. Save the time, search term, and link as well.
  6. Load a "block template", which is an HTML file with replaceable sections.
  7. Package the "block", time, search term, and link into the "block package".
  8. Repeat as many times as nessesary to make packaged blocks.
  9. Load a "scarlet rat template", another HTML file with replacable sections. This text is written in the template!
  10. Organize all the blocks by time, and stuff them into the "scarlet rat template".
  11. Save to the main HTML file. Update github using .bat file.
